# AI-ImageRecognition-Project 4

## 📌 Project Description
This project demonstrates the use of a **pre-trained deep learning model** to perform **image recognition**.  
The system takes an input image and predicts what object is present in the image.

---

## 🚀 Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pillow (PIL)

---

## 🤖 Model Used
We used **MobileNetV2**, a pre-trained model provided by TensorFlow and trained on the ImageNet dataset.
